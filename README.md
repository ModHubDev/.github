## About Us
**ModHub.gg** is an organization focused on mods and user‑generated content (UGC) for video games. Our goal is to bring creators and players together and provide a seamless, safe, and modern ecosystem for discovering, sharing, and managing mods.

**What We Build**
- Web platform for browsing, publishing, and rating mods.
- SDKs & integrations for engines (Unity, Unreal, APIs).
- Community tools including moderation, reporting, and guidelines.

## Projects (Example)
- frontend — React + Vite client.
- api — Authentication, moderation, mod storage.
- sdk-* — Client/server SDKs.

## Tech Stack
- Frontend: React, Vite, Shadcn, TypeScript.
- Backend: AspCore, PostgreSQL, Redis.
- Infra: Docker
- Analytics: OpenTelemetry, Prometheus, Grafana.

## Roadmap
